## 1. Objetivo de la práctica
El objetivo de esta práctica es que como alumno instale el lenguaje de programación Haskell y el sistema de control de versiones Git, configure el entorno de desarrollo y cree e inicialice un repositorio en GitHub. Con ello se busca familiarizarme con las herramientas básicas que se utilizarán durante el curso y comprender el flujo del trabajo inicial.
## 2.Tiempo requerido para realizar la práctica
El tiempo aproximado que me tomó realizar la práctica fue de entre 2 y 3 horas, ya considerando la instalación de Haskell, Git, la configuración del entorno y la creación del repositorio.
## 3.Comentarios y problemas encontrados durante la instalación
Durante la instalación no se presentaron problemas significativos, ya que el proceso fue bastante sencillo al realizarse en el sistema operativo Windows. Inicialmente se tenía la intención de trabajar en Linux, sin embargo, al estar en proceso de instalación resultó más complicado, por lo que se decidió continuar en Windows. La mayor dificultad se presentó en el uso de GitHub, ya que al inicio no se comprendía completamente su funcionamiento. Esta situación se resolvió revisando la documentación oficial y siguiendo las indicaciones proporcionadas en el laboratorio.

*The Glorious Glasgow Haskell Compilation System, version 9.6.7*
