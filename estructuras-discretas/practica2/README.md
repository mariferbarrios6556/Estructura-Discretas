Materia: Estructuras Discretas
Semestre: 2026-II

## Objetivo de la práctica

El objetivo de esta práctica es reforzar los conceptos básicos del lenguaje Haskell mediante la implementación de funciones que emplean tipos primitivos, tuplas, listas por comprensión, guardas, condicionales y operaciones a nivel de bits. Asimismo, se busca familiarizar al estudiante con el uso del intérprete GHCi para compilar y ejecutar programas funcionales.

 ## Tiempo requerido

El tiempo estimado para la realización de la práctica fue de aproximadamente 2 horas, considerando:

Desarrollo de las funciones.

Pruebas en GHCi.

Corrección de errores de compilación.

Elaboración del archivo README y evidencia de ejecución.

## Comentarios adicionales

Durante el desarrollo se presentaron errores relacionados con la ubicación de las sentencias import, los cuales fueron corregidos asegurando que se encontraran al inicio del archivo, como lo establece la sintaxis de Haskell.

Las funciones fueron probadas individualmente en GHCi para verificar su correcto funcionamiento antes de la entrega final.
