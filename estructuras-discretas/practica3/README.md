# Objetivo de la práctica

El objetivo de esta práctica es comprender el uso de clases de tipo en Haskell, la creación de tipos de datos propios y el uso de listas por comprensión para generar y filtrar listas.

Además se busca practicar la definición de funciones simples respetando las restricciones del lenguaje y utilizando únicamente operaciones básicas.
